## Architecture

A stand alone back end servicing a static front end. This server fields requests from a client and responds with data collected from various APIs.
  - Language:
    - Javascript
  - Libraries:
    - Express.js
    - Dotenv
    - Cors
  - Tools and frameworks:
    - Node.js
    - Heroku
    - PostgreSQL
